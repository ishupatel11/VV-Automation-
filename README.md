# v.v automation Website

A modern, responsive industrial automation website built with HTML5, CSS3, and JavaScript.

## Features
- **Modern Dark UI**: Premium industrial aesthetic with glassmorphism effects.
- **Responsive Design**: Fully functional on mobile, tablet, and desktop.
- **Interactive Elements**: Smooth scrolling, sticky header, and scroll animations.
- **High-Quality Imagery**: Integration with Unsplash for professional industrial visuals.

## How to Run
Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Custom Properties / Variables)
- Vanilla JavaScript
- Font Awesome (Icons)
- Google Fonts (Manrope & Space Grotesk)
